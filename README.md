# Hi there 👋 I'm Tony Tran

I’m an **MS (Thesis) student in Engineering Data Science & AI at the University of Houston** and a **Graduate Research / Teaching Assistant** in the **[Network Autonomous and Intelligent Learning (NAIL) Lab](https://binhu85.github.io/team-members/)**.  
My work focuses on **TinyML / EdgeAI**, specifically **hardware-aware neural architecture search (NAS)** and **efficient object detection** on microcontrollers.

- 🔭 **Currently working on:** Energy-Efficient Perception System, RL-Guided Iterative NAS
- 🧠 **Interests:** Efficient NAS, Once-For-All Models, Pruning/Quantization, Robust CV, Embedded AI Deployment
- 🏫 **Teaching:** AI in Robotics, Embedded Systems, Sensor Applications, Intro to C++
- 🏅 **Credential:** Engineering in Training (FE/EIT) License in Electrical and Computer (Texas: 81224)
